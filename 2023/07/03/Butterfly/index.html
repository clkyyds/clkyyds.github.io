一、Butterfly 快速开始

安装

在hexo根目录下

应用主题

修改 Hexo 根目錄下的 _config.yml，把主題改為 butterfly（这个名字是针对你主题具体的文件名）

安装插件

如果你沒有 pug 以及 stylus 的渲染器，請下載安裝：

二、Butterfly 主题页面

Front-matter

Front-matter 是 markdown 文件最上方以 — 分隔的區域，用於指定個別檔案的變數。

  • Page Front-matter 用於頁面配置
  • Post Front-matter 用於文章頁配置

标签页

  • 前往你的 Hexo 博客的根目錄
  • 輸入 hexo new page tags
  • 你會找到 source/tags/index.md 這個文件
  • 修改這個文件：記得添加 type: “tags”

分類頁

  • 前往你的 Hexo 博客的根目錄
  • 輸入 hexo new page categories
  • 你會找到 source/categories/index.md 這個文件
  • 修改這個文件：記得添加 type: “categories”

三、Butterfly 主题页面

网站资料

修改站點配置文件 _config.yml（_config.yml的文件是指butterfly主题里面的文件）

導航欄設置

参数 解释
logo 網站的 logo，支持圖片，直接填入圖片鏈接
display_title 是否顯示網站標題，填寫 true 或者 false
fixed 是否固定狀態欄，填寫 true 或者 false
